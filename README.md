Typed client SDK for connecting Next.js apps to Based CMS realtime content.

## Install

```bash
pnpm add cms-client convex
```

## Environment

```bash
BASED-CMS-SLUG=my-project
BASED-CMS-KEY=bcms_test-...
```

- `BASED-CMS-SLUG`: public project slug
- `BASED-CMS-KEY`: `bcms_<test|live>-<base64(deploymentName.SECRET)>`

## New Project?

Try our CLI for your first project.

```bash
npx @based-cms/get-started my-app
```
